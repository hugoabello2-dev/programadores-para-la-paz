Repositorio

Un repositorio es un espacio donde se almacenan y organizan los archivos de un proyecto de software. Allí se guardan las versiones, cambios y documentos relacionados con el desarrollo. Herramientas como GitHub permiten compartir repositorios y trabajar en equipo de manera organizada.

Commit

Un commit es un registro de los cambios realizados en un proyecto. Cada vez que se hace un commit, Git guarda una versión específica del trabajo junto con un mensaje descriptivo. Esto permite llevar un historial ordenado de las modificaciones realizadas.

Push

Push es el comando que permite enviar los cambios guardados en Git desde el computador local hacia un repositorio remoto en GitHub. Gracias al push, los archivos y actualizaciones quedan publicados en línea y disponibles para otros colaboradores.
